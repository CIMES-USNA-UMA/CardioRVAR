# CardioMVAR is an R package designed for the analysis of open and, mainly, closed-loop interactions that exist between cardiovascular variables. 
# This project is in development by the University of Malaga by medical student Alvaro Chao-Ecija, under the supervision of Professor Marc Stefan Dawid
# Milner. With this contribution to the R statistical database, users are allowed to analyze closed-loop interactions, estimate baroreflex sensitivity
# indexes, and evaluate causal couplings between the variables. This package is being developed with a separate Shiny interface, CardioMVAR Shiny, for
# a more user-friendly experience.
